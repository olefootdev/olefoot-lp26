Referee, commentator, and tactical advisor integrated frame-by-frame into the simulation loop. Resolves shots, builds story arcs, decides penalties. Powered by Anthropic Claude via Railway API.

4.3 · The Architectural Decision

AI at creation, not at runtime.

Most AI-native games generate responses at runtime: every player action, every match event, every piece of narrative costs an API call. As the user base grows, so does the AI bill — often exponentially. This is the "AI scaling trap" that has broken several well-funded startups.

Olefoot inverts this model. Claude is used once per player creation — to generate the player's DNA: attributes, behavioral traits, personality, Signature Moves, narrative backstory. Once generated, this DNA is stored in Supabase and consumed by the match engine with no additional API calls.

Cost of AI per match played
$0
After initial player DNA generation
1 match vs 1,000,000 matches
Identical
Marginal cost is invariant — this is the architecture that scales.

5.A · How Legacy Mode Was Built

What if a real player
could exist inside the game?

Legacy Mode was born from a simple question: what if a real footballer could exist inside the game? In Olefoot, any professional player can be transformed into a digital card with their own DNA — real attributes, career history, playing style. But we went further.

With the advancement of AI agents, each card now carries an autonomous agent that makes decisions inside the match based on the real player's profile. A Diego Lugano in Legacy Mode is not just a card with 90 tackling. It is an agent that organizes the defensive line, that activates Captain Influence when morale is high, that executes aerial duels at the right moments — because the agent was trained with that player's tactical DNA.

Appendix C · Glossary

Terms used in this whitepaper.

GameSpiritThe AI narrative engine that resolves match events (goals, penalties, red cards) and generates real-time commentary via Claude.
SmartFieldPython-generated tactical geometry: zones, positional anchors, radius of influence. Pre-computed and served as JSON — zero runtime AI cost.
Yuka AgentsIndependent steering-behavior AI agents for each of the 22 players on the field. Each agent reads DNA and field context to make autonomous decisions.
DNAPlayer behavioral profile generated once by Claude: attributes, traits (riskTaking, buildUpPreference), Signature Moves, and narrative backstory.
Legacy ModeThe system for transforming real professional players into digital cards with autonomous AI agents trained on their tactical DNA.
EXPLifetime, non-purchasable experience points that determine career tier progression for the manager.
Market MakerThe system that guarantees instant liquidity for player card sales by providing dynamic buy offers based on position and OVR rating.
OVROverall Rating — composite metric calculated from player attributes (pace, finishing, passing, tackling, vision, strength).
